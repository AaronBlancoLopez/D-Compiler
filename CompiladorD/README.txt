
COMPILADOR PARA LENGUAJE D
____________________________________

Como usarlo:
  - Desde la terminal acceder a la carpeta del proyecto
  - Ejecutar el comando 'make'
  - Ejecutar el comando './compilar [ruta al fichero fuente]'


Información de uso:
    Las instrucciones funcionarán para cualquier sistema operativo UNIX-like.

NOTA:
    En caso de tener algún problema a la hora de generar el ejecutable mediante 'make', también se puede compilar el proyecto ejecutando 'gcc *.c', y ejecutarlo mediante './a.out [ruta al fichero fuente]'
